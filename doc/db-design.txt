=============== SPECIFICATION =================
Consumer:
    - Manage Balance
        - if (Balance < Price) {
            reject Transaction
        } else {
            approve Transaction
            Balance -= Price
        }
        - Balance is shared among children
        - Consumer app able to increase or decrease balance
    - Manage Children
        - Consumer can create and delete children
    - Manage Transaction
        - All Transaction (approve or reject) will notify Consumer

Merchant:
    - Manage Product 
        - Create Product
        - Delet Product
    - Manage Transaction
        - Create Transaction  
        - Cancel Transaction
    - Manage settlement
        - Received payment from settlement
    - UI Page
        - Home -> List of product
        - History -> List of Transaction
        - Account -> all money related

Admin:
    - Manage User 
        - All CRUD operation
    - Manage Merchant
        - All CRUD operation
    - Manage Settlement
        - Do settlement for all merchant
    - Manage Balance
        - All CRUD Operation

==================== MODEL ====================
Transaction:
    - Invoice Number
    - Amount
    - Timestamp
    - Merchant
    - Consumer
    - Child
    - Settlement State ? Close : Open 

Settlement:
    - Transaction[]

Consumer:
    - Name
    - Address
    - Phone
    - Balance
    - Account Number
    - Child[]

Merchant:
    - Name
    - Address
    - Phone
    - Account Number
    - product[]      

Product:
    - Id
    - Name
    - Price 
    - Picture
